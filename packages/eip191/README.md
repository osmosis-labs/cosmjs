# @cosmjs/eip191

[![npm version](https://img.shields.io/npm/v/@cosmjs/eip191.svg)](https://www.npmjs.com/package/@cosmjs/eip191)

Helpers for EIP191 based signing which are shared between @cosmjs/launchpad and
@cosmjs/stargate.
